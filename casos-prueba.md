# Casos de Prueba - Sistema de Ventas

## Caso 1: Agregar producto correctamente
**Entrada:** Nombre y precio válido  
**Resultado esperado:** Producto agregado sin errores

## Caso 2: Precio inválido
**Entrada:** Texto en lugar de número  
**Resultado esperado:** Mensaje de error

## Caso 3: Ver total sin productos
**Entrada:** Opción "Ver total" sin productos registrados  
**Resultado esperado:** Mensaje indicando que no hay productos

## Caso 4: Salir del sistema
**Entrada:** Opción "Salir"  
**Resultado esperado:** Programa finaliza correctamente
